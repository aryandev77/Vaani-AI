# **App Name**: Vaani AI

## Core Features:

- Real-Time Voice Translation: Translates voice conversations in real-time between different languages.
- Cultural Context Engine: Uses AI tool to analyze the cultural context of phrases and translates them accurately, preserving the original intent and meaning.
- Emotion and Tone Preservation: Detects the user's tone and emotion to ensure the translated phrase conveys the same sentiment in the target language, powered by an AI tool.
- Real-Time Voice with Lip Sync: Matches the translated audio with lip movements in real-time, enhancing the user experience and comprehension.
- Learning Mode: Provides cultural insights and explanations after conversations, helping users learn about different cultures and the meanings behind idioms.
- User Authentication: Securely manage user accounts, profiles, and subscription status
- Firestore Integration: Stores user conversations and cultural phrase data for future reference.

## Style Guidelines:

- Primary color: HSL 210, 100%, 50% (RGB: #007BFF). A professional and calming blue, similar to Google and Samsung, conveying trust and intelligence.
- Secondary color: HSL 0, 0%, 100% (RGB: #FFFFFF). Clean white for backgrounds and content containers to ensure clarity and readability.
- Accent color: HSL 120, 60%, 40% (RGB: #3BB143). A subtle green to indicate successful translations or cultural matches, promoting user confidence.
- Headline font: 'Google Sans' (or 'Roboto'), a clean and modern sans-serif for titles and headings to maintain a professional appearance.
- Body font: 'Roboto' (or 'Noto Sans'), a readable and accessible sans-serif for body text and descriptions, ensuring a comfortable reading experience.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use minimalist and easily recognizable icons with rounded edges. Match the icon color to the primary blue for a cohesive look.
- Employ a card-based layout with subtle shadows for information blocks. This provides a structured and clean interface, improving the user experience.
- Use smooth transitions and micro-interactions to provide subtle feedback during translations. Aim for a fluid, non-jarring experience similar to high-end modern interfaces.
